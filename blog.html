Question-2: What is the difference between innerHTML, innerText, and textContent ?
-> innerHTML is used to insert HTML elements inside an element.
-> innerText is used to visible only text and ignore hidden elements.
-> textContent is used to show all text (including hidden elements)
Question-3: What is event delegation in the DOM?
Event delegation event handle korar ekta efficient way. Jodi amader onek gulo similar element thake (jemon list er item), tahole proti ta element er jonno alada event listener add na kore, amra shudhu matro parent element er upore ekta event listener add korte pari. Tarpor event jokhon hoy, event.target diye amra check korte pari kon specific element e event ta hoyeche.
Question-4: What is event bubbling in the DOM?
Event bubbling holo JavaScript er ekta mechanism jekhane kono child element e ghote jawa event tar parent element gulor modhye uporer dike choriye pore. Er maney, jodi kono child element e event ghote (jemon click), tahole sei event ta tar parent, grandparent, aro uporer element gulokeu trigger korbe.
Question-5: How do you create, add, and remove elements using JavaScript?
1. Create an element: document.createElement()
2. Add an element: parent.appendChild()
3. Remove an element: parent.removeChild()
